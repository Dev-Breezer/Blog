# Java 代码块

&emsp;

#### 在 Java 当中代码用{}括起来的叫做代码块  
#### 出现在方法中的代码块我们叫做**普通代码块**，普通代码块与一般语句执行顺序相同（即：按照顺序执行）
#### 出现在类中的代码块我们叫**构造代码块**

&emsp;

### 构造代码块与普通代码块的区别：

&emsp;

> + 普通代码块：顺序执行，先出现，先执行
> + 构造代码块：创建对象时调用，优先于构造方法执行
> + 静态代码块：静态代码块会先于构造方法先于构造代码块先于普通代码块执行

&emsp;

### 当某个类实例化对象后，无论实例化对象的个数，静态代码块总是执行一次，每一次实例化对象，构造代码块和构造方法都会执行

&emsp;

### 注意：

&emsp;


> + 构造代码块中，可以给当前类中的成员属性和静态属性赋值
> + 静态代码块中，只能给当前类中的静态属性赋值

```java
    // 静态代码块
    static{
        System.out.println("我是静态代码块2");
        price = 3000;
        Cat temp = new Cat();
        temp.name = "小七";
    }

      //  构造代码块
    {
        System.out.println("我是构造代码块1");
        price = 200;
        month = 5;
    }
```
